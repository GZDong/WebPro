• css使用元素选择器+{ xxx : xxx;...}来标记所有这类元素的样式
    h1,h2,h3,h4,h5,h6 {
    color: green;
    }
  • css使用#+id+{ xxx : xxx;...}来标记这个id指定的元素的样式
    #red {color:red;}
  • css使用#+id 内部元素+{ xxx : xxx;...}来标记这个id内部的该类元素的样式
    #sidebar p {
    font-style: italic;
    text-align: right;
    margin-top: 0.5em;
    }
  • css使用.+class+{ xxx: xxx;...}来标记所有拥有该类的元素的样式
    .center {text-align: center}
  • css使用.+class 内部元素+{ xxx:xxx;...}来标记所有拥有该类的元素的样式
    .fancy td {
    color: #f60;
    background: #666;
    }
  • css使用 内部元素+.+class+{ xxx:xxx;...}来标记拥有该类名的元素的样式
    td.fancy {
    color: #f60;
    background: #666;
    }
  • css使用 [属性] / [属性=xxx] {xxx:xxx;...}来指定拥有该属性或者该属性=xxx的元素的样式
    [title]
    {
    color:red;
    }
